# 録画機器・録画文化の衰退

## 概況

2010年代後半から、日本における BD/HDD レコーダー市場は急速に縮小している。BDZ-EW510 のようなサポート終了機器が増加する中、録画文化そのものが構造的な転換点を迎えている。

## 各メーカーの撤退状況

### 市場規模の推移

| 年 | 国内出荷台数 | 備考 |
|---|---|---|
| 2011年 | **約 639 万台** | ピーク |
| 2025年 | **約 62 万台** | ピーク比 約 10% |

10年強で市場が 10分の1 以下になった。

### メーカー別撤退タイムライン

| メーカー | 状況 | 時期 | 備考 |
|---|---|---|---|
| **三菱電機** | 撤退済み | 2010年代中頃 | 比較的早期に撤退 |
| **日立** | 撤退済み | 2010年代 | パナソニックとの合弁（P&H）を経て撤退 |
| **パイオニア** | 撤退済み | 2010年代前半 | 民生用レコーダーから早期撤退 |
| **シャープ** | 自社開発撤退 | 2024年 | 2024年6月（2K）・11月（4K）にパナソニック OEM へ切替。ブランドは存続 |
| **TVS REGZA（東芝）** | 撤退 | **2026年1月** | 製造パートナーの船井電機が2024年10月破産 → 強制終了 |
| **Sony** | 撤退 | **2026年2月** | 最終出荷完了。2003年に世界初の民生用 BD レコーダーを発売したメーカーが撤退 |
| **パナソニック（DIGA）** | **継続中** | — | 唯一の自社開発継続メーカー |

### 各撤退の背景

**Sony（2026年2月撤退）:**
- 2003年に世界初の民生用 BD レコーダーを発売した張本人
- BD 規格策定に深く関与したメーカーが最後に市場を去るという皮肉
- 2024年7月: 光学メディア部門で 40% の人員削減
- 2025年1月: 録画用 BD・MiniDisc・カセットの生産終了を発表
- 2026年2月: 最終出荷完了

**TVS REGZA（東芝・2026年1月撤退）:**
- 2024年10月に製造・開発パートナーの**船井電機が破産**
- これが決定打となり強制的に生産終了
- 撤退は計画的なものではなく、パートナー企業の経営破綻による突然の終了

**シャープ（2024年〜 OEM 化）:**
- 自社開発は終了しパナソニックからの OEM 供給に切り替え
- 「AQUOS ブルーレイ」ブランドは継続しているため、消費者からは撤退に見えにくい
- Sony の撤退報道の際、シャープは「シャープにはBDレコーダーがあります」と公式コメントを発表（OEM だが）

**パナソニック（DIGA）:**
- 現時点で唯一の自社開発継続メーカー
- 事実上の「最後の砦」状態

## 市場縮小の要因

### ストリーミングサービスの台頭

| サービス | 特徴 |
|---|---|
| Netflix / Amazon Prime Video | 見たいときに見る。録画不要 |
| TVer（テレビ局公式） | 無料見逃し配信（1週間程度） |
| NHK+ | NHK 番組の見逃し配信 |
| U-NEXT / Hulu 等 | 月額課金で豊富なコンテンツ |

「録画して後で見る」という行動が「ストリーミングで見る」に代替されつつある。特に若年層においてこの傾向が顕著。

### 生活スタイルの変化

- スマートフォン・タブレットでの視聴が増加 → テレビ中心の生活が変容
- 「放送に合わせてテレビの前に座る」という習慣の消滅
- 録画番組を「後で見る」文化自体が縮小

### テレビ離れ

- 特に 20〜30代のテレビ視聴時間が激減
- テレビ自体を持たない世帯が増加
- 「テレビを録画したい」という需要の母数が減少

## 録画文化の技術的な矛盾

录画技术の进化が皮肉にも録画文化の衰退を招いた側面がある。

### コピー制限の強化が自滅を招いた

```
1999年: BSデジタル放送開始 → コピーワンス導入
2004年: 地上デジタル放送開始 → コピーワンス継続
2008年: ダビング10 導入 → 多少緩和
2012年: DVD リッピング禁止（著作権法改正）

→ 録画したコンテンツをどこでも見られるわけではない
→ ストリーミングの方が手軽という逆転が発生
```

**コピー制限によって録画コンテンツの利便性が低下し、ユーザーがストリーミングに流れた。**

### BD レコーダーの複雑化

- DTCP-IP、DLNA、ダビング10、HDCP…と規格が増え、普通のユーザーには難解に
- 「ただ見たいだけ」のユーザーには Netflix の方がずっと簡単

## デジタルアーカイブとしての録画の意義

録画文化が衰退する中でも、録画が持つ本来の意義は失われていない。

### 放送の一回性

- テレビ放送は本来「一度きり」のメディア
- ストリーミングでは配信されない番組（地方ローカル番組、ニュース速報、スポーツ中継等）
- 権利関係で配信に乗れない番組（音楽著作権、映像使用権等）

### ストリーミング配信の限界

| 問題 | 内容 |
|---|---|
| 配信終了リスク | 権利切れやサービス終了でコンテンツが消える |
| 地域制限 | 日本のサービスは日本国内のみ |
| 全番組は配信されない | 地方局・古い番組・一部スポーツ中継等 |
| オフライン視聴の制限 | インターネット接続が必要（ダウンロード機能あっても暗号化） |

### BDZ-EW510 のようなケースが示すもの

- 「自分が録画した番組を、機器のサポート終了後にどう保存するか」という問題
- 法律・技術・ビジネスモデルが複合して、**ユーザーが自分のコンテンツにアクセスできない**という矛盾
- デジタルコンテンツの「所有」とは何か、という根本的な問い

## 録画文化の「ガラパゴス化」と海外勢の不在

### テレビ録画は日本固有の文化

テレビの「録画して後で見る」という文化は、実は**日本で特に発達した文化**である。

| 地域 | 録画文化 | 背景 |
|---|---|---|
| 日本 | 非常に強い | HDD/BD レコーダーが普及。「タイムシフト視聴」が一般的 |
| 北米 | TiVo 等の DVR が存在したが衰退 | ケーブル TV 会社の STB に内蔵 → ストリーミングへ移行 |
| 欧州 | 限定的 | STB 録画が主流 |
| その他 | 概して限定的 | — |

日本において録画文化が特に発達した理由：
- 地上波・BS・CS と放送の種類が多い
- NHK を含む多チャンネル環境
- アニメ・ドラマを全話録画する視聴習慣
- 「見逃し配信」が整備される前は録画が唯一の手段だった

### 海外の強力なソフトウェア会社が参入しない理由

VLC・Plex・Kodi・Emby など、海外発の強力なメディアソフトウェアは世界中で使われているが、**DTCP-IP には対応していない**。

その理由：

| 理由 | 詳細 |
|---|---|
| **市場が日本限定** | DTCP-IP を必要とする市場がほぼ日本のみ。グローバルな開発リソースを投入するインセンティブがない |
| **ライセンスコストが高い** | DTLA ライセンスが必要で、OSS プロジェクトには不可能 |
| **仕様が非公開** | 完全仕様書は契約者のみ。エンジニアが実装しようにも情報が手に入らない |
| **法的リスク** | DRM 回避として訴訟リスクがある |

結果として、日本独自の DRM エコシステムに、日本の専業ベンダー（DigiOn 等）だけが対応し、**グローバルなソフトウェアエコシステムから完全に孤立した閉じた世界**が生まれた。

### 孤立の代償

- **競争がない** → DiXiM 等の価格が高止まりし、品質向上も緩やか
- **OSS による革新がない** → Kodi や Plex のような進化が起きなかった
- **ユーザーの選択肢が極めて少ない** → 数少ない日本製有料ソフトに依存するしかない
- **機器のサポート終了 = 詰み** → 対応ソフトが無くなれば録画コンテンツにアクセスできなくなる

これは DTCP-IP という規格が「日本の録画文化を守るため」に設計されたにも関わらず、その閉鎖性がかえって日本の録画文化の弱体化を招いたという、もう一つの皮肉である。

## 今後の展望（推測）

### 短期（〜2030年）

- BD レコーダー市場は Panasonic・Sharp（OEM）が支えるが、縮小は不可避
- ストリーミングサービスの見逃し配信期間が延長される動き（TVerは7日→longer）
- DTCP-IP 対応機器の新規開発は減少

### 中期（2030〜2035年）

- 地上デジタル放送のコピー制限緩和議論が再燃する可能性
  - 理由: 録画機器が絶滅に近づき、コピー制限を維持する意味が薄れる
- ストリーミングが放送を完全に代替する可能性（放送法改正の議論）
- 古い録画機器で保存されたコンテンツが再生不能になるケースが社会問題化するかも

### 長期（2035年以降）

- 「テレビ録画」という概念自体が消滅する可能性
- 放送 → ストリーミング のパラダイムシフト完了
- DTCP-IP は歴史的な技術として記録されるのみ

## まとめ

BDZ-EW510 のサポート終了は、単一機器の問題ではなく、**録画文化そのものの終焉**の一部である。

技術的・法的制約（DTCP-IP、著作権法）によってユーザーが自分の録画コンテンツを自由に扱えない状況は、皮肉にもストリーミングへの移行を加速させた。その結果として録画文化自体が衰退し、録画機器メーカー（Sony のようなレコーダー事業撤退）にとっても市場が消滅するという自業自得の側面がある。

今 BDZ-EW510 の録画を保存しようとする行為は、**デジタル時代のアーカイブ活動**として意義深いが、技術と法律の両面から多くの障壁に直面しているというのが現状である。

## DTCP-IP への批判的考察

> 以下は客観的な事実ではなく、この調査を通じた個人的な分析・意見である。

### 設計思想が時代と合わなくなった

DTCP-IP が設計された 1990年代末〜2000年代初頭、脅威モデルは「インターネットを使った大規模な海賊版流通」だった。当時は正当な懸念だったが、**「自分が合法的に録画したコンテンツを、自分の機器が壊れる前にアーカイブしたい」という個人ユーザーのユースケースは、設計に組み込まれなかった。**

四半世紀後の今、その設計判断のコストをエンドユーザーが払っている。

### ライセンスモデルの根本的な欠陥

| 問題 | 詳細 |
|---|---|
| **個人・研究者向けパスが存在しない** | Small Adopter でも企業向け。Evaluator 枠は「製品出荷禁止」で個人の問題解決に使えない |
| **年間費用モデル** | 大量生産メーカーには合理的だが、個人・小規模開発者には原理的に不合理 |
| **OSS 化の余地がない** | DTLA 証明書が AKE に必須のため、ライセンス取得者以外は原理的に正規実装できない |
| **仕様書さえ非公開** | 実装詳細を含む完全仕様書は契約者のみ。「学習・研究目的」でのアクセスすら不可 |

比較として: Apple は iTunes の DRM（FairPlay）を 2009年に撤廃した。ユーザーが「自分で購入した音楽を自分で使えない」という矛盾に向き合った結果だった。DTCP-IP にはその判断がなかった。

### 「もし違っていたら」という問い

- **個人開発者向け低廉ライセンス**（例: 非商用・年間 $100 程度）があれば → OSS エコシステムが育ち、DiXiM のような専業ベンダーだけでなく多様なツールが登場していた可能性がある
- **DTCP-IP なしの設計**だったら → 利便性が高く、録画文化はより長く続き、ストリーミングとの共存もあり得た
- **ダビング10 と同様の「合理的な緩和」** があれば → ユーザーと権利者の信頼関係を維持しつつ、正当なアーカイブ活動が可能だった

### 自滅のロジック

```
コピー制限を強化・ライセンスを高額維持
        ↓
録画コンテンツの利便性が下がる（機器ごとに縛られる）
        ↓
ユーザーがストリーミングに流れる（Netflix 等は DRM なしで使いやすい）
        ↓
録画機器の需要が急減
        ↓
メーカーが撤退（Sony 2025年にレコーダー事業終了）
        ↓
DTCP-IP が守るべき市場そのものが消滅
```

**権利保護の強化が、皮肉にも権利者と市場の双方を傷つけた**、と言えるかもしれない。

### ユーザー視点での不満の正当性

この調査の発端となった問題——「自分が録画した番組を、機器のサポート終了後に保存できない」——は、技術的・法的には整合しているかもしれないが、**道義的には非常に問題がある**。

- ユーザーは受信料（NHK）を払い、機器を買い、合法的に録画した
- その録画を自分のストレージに移す行為が、ライセンスの壁で阻まれる
- 機器メーカーはサポートを終了できるが、コンテンツは人質のまま

「自分のコンテンツを自分で使えない」という状況は、デジタル所有権の根本的な矛盾であり、この問題は DTCP-IP に限らず、デジタルコンテンツ全般の課題でもある。
