# 代替アプローチ

## HDMI キャプチャー（アナログホール）

### 概要

BDZ-EW510 で再生しながら、HDMI 出力をキャプチャーカードで録画する方法。「アナログホール」と呼ばれる手法の現代版。

### 必要なもの

- HDMI キャプチャーカード（外付け USB 型 or PCIe 型）
- PC（録画先）または スタンドアロン録画機器

### 推奨キャプチャー機器

| 製品 | 解像度 | 備考 |
|---|---|---|
| AVerMedia LIVE GAMER EXTREME 3 | 4K/30fps または 1080p/60fps | 定番。PCIe 型 |
| AVerMedia GC551G2 | 4K/30fps + 4K パススルー | 外付け USB 型 |
| I-O DATA GV-US2C/HD | Full HD + 4K/60p パススルー | 外付け USB 型 |
| **I-O DATA GV-HDREC/E** | 1080/60p | **PC 不要**。外付け HDD に直接録画 |

### 法的地位

| 状況 | 法的評価 |
|---|---|
| 地上デジタル放送をリアルタイムキャプチャー | **合法**（放送自体は DRM なし） |
| BDZ-EW510 の録画を**再生しながら**キャプチャー | グレーゾーン（DTCP-IP 保護の「出力」をキャプチャーする行為） |
| DTCP-IP 保護を「回避して」キャプチャー | 違法 |

BDZ の HDMI 出力には **HDCP（HDMI の DRM）** が付いている可能性があり、HDCP 対応キャプチャーカードでないと録画できない場合がある。ただし HDCP 非対応の古いキャプチャー機器だと通ることも。

### 品質

- **再エンコードが発生する**: 元のMPEG-2データではなく、キャプチャーカードが再圧縮（H.264等）するため若干の画質劣化
- **ビットレート次第で実用上は問題なし**: 1080p/20Mbps 以上なら放送品質と遜色ない
- **真のロスレスキャプチャーは不可能**（HDMI 信号自体が既に圧縮済み）

### メリット・デメリット

| メリット | デメリット |
|---|---|
| 機器や DRM に依存しない確実な方法 | 等速キャプチャー（1時間番組を1時間かけて録画） |
| ソフトウェア不要の機器もある | 若干の画質劣化 |
| 追加ソフト費用不要（ハードウェアのみ） | 別途キャプチャー機器が必要（¥10,000〜） |

## Sony nasne（ネットワークレコーダー）への移行

### nasne とは

従来の BD レコーダーと異なり、ネットワーク配信に特化した録画機器。

| 項目 | BDZ-EW510 | nasne |
|---|---|---|
| 主な用途 | 物理メディア再生・録画 | ネットワーク配信 |
| PC/スマホ連携 | 限定的 | 優秀（アプリ多数） |
| DLNA/DTCP-IP | 対応 | 対応（より充実） |
| BD ドライブ | あり | なし |
| 現在の状況 | 旧機種（サポート終了） | **Sony 版は販売・サポート終了** |

### nasne の現状（2026年）

- **Sony 製 nasne**: 2019年に販売終了。2024年7月にアフターサービス終了。**2027年7月にネットワーク機能が完全停止予定**
- **Buffalo 製 nasne**: Buffalo が nasne ブランドを引き継ぎ販売継続中。Sony 製 nasne からのデータ移行機能（「引越しダビング機能」）あり

→ **BDZ-EW510 から Buffalo nasne への移行は現実的な選択肢**。ただし nasne の方が BDZ-EW510 より PC 連携が優れているため、問題が解決しない可能性もある（nasne も DTCP-IP 保護のまま）。

## RECBOX（NAS）への一時退避

詳細は [06_existing_solutions.md](./06_existing_solutions.md) を参照。

要点:
- BDZ-EW510 → RECBOX へのダビングは可能
- ただし DTCP-IP 保護は維持されたまま
- PC での視聴には引き続き DiXiM 等が必要

---

## BD メディアへのダビング（物理的手段）

```
BDZ-EW510 → BD-R/BD-RE → 別の BD レコーダーでコピー
```

- ダビング10 の範囲内であれば合法
- DTCP-IP が不要になる（物理メディアは AACS が保護するが、BD レコーダーで通常再生可能）
- BDZ-EW510 の BD ドライブがまだ動作している間に実施する必要がある
